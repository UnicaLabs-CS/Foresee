Foresee
=======

This is a Java porting of the original ART C++ framework from the University of
Cagliari.

## Structure

```
|- README.md
|- .gitignore
|- .idea/
|- it/unica/foresee/
|   |- predictions
|   |- tests
|- docs/
```

## Requirements
- Java JDK (Tested on Oracle Java8)
- jUnit 4
- Movielens dataset
