Foresee
=======

This is a Java porting of the original ART C++ framework from the University of
Cagliari.

## Structure

```
|- README.md
|- .gitignore
|- .idea/
|- it/unica/foresee/
|   |- datasets/
|   |- interpreters/
|   |- tests/
|   |- predictions/
|   |- Foresee.java
|
|- docs/
|- libs/
|- makefiles/
|- test-data/
```

## Requirements
- Java JDK (Tested on Oracle Java8)
- Movielens dataset
